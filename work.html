IAT334 Interface Design: Final Project

For my school course "Interface Design", I worked with three other students to create a user interface prototype for bulk purchases of groceries among neighbors.

Week 1: Ideating the Application

My team and I brainstormed and came up with three ideas of the application's function.
We focused on the domain of shopping for all our ideas, which included virtual tour shopping, fitting room at home, and wholesale delivery.

Week 2: User Personas

After deciding on the idea that we will proceed with to be wholesale delivery, we developed two user personas that identifies potential users of this application.

Week 3: Grey Box Prototype

In Week 3, we designed the basic interface of the application as a grey box prototype, outlining the basic functionalities of the application.

Week 4: Style Guide

Style Guide for the project application was designed in Week 4, as my team and I settled with specific colors and fonts that will be used to give the application an identity.
Some aspects of the style guide and photos from web were applied to the grey box prototype.

Week 5: Prototype Interaction

The design of the application was finalized and the prototype was made interactive.

Week 6: User Testing

The prototype was user tested and iterated for the final submission.
